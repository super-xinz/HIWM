# Archived upstream documentation

The `openavatarchat/` directory contains documentation and community material inherited from the base OpenAvatarChat checkout. It is retained for provenance and compatibility reference, excluded from the current VitePress site, and must not be presented as this project's community, release history, online demo, or official support channel.

For current project documentation, use `docs/guides/`, `docs/research/`, and `docs/reports/`. For authoritative upstream material, visit [HumanAIGC-Engineering/OpenAvatarChat](https://github.com/HumanAIGC-Engineering/OpenAvatarChat).
